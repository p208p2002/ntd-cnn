# NTD-CNN
新台幣鈔票辨識
> 圖像識別課堂作業

## Methods
- convolutional neural network
- data augmenting
- dropout

## Dataset
100、500、1000元鈔票各20張
### Train setting
100、500、1000元鈔票各10張
### Test setting
100、500、1000元鈔票各10張
